Kent Simple Fancy Gallery describes itself. Just install and activate and paste shortcode anywhere on your page to create a fancy gallery.

Plugin Feature

+ Simple and Easy to use
+ CSS3 Animation
+ Shortcode system

== Installation ==

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After Activate Visit your Single Post.
4. and after activate you will see menu on dashboard "Kento PVC" will get stats.
